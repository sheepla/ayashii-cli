### Depandences

- curl: a url transfer
- [octobanana/euarel](https://github.com/octobanana/euarel) : a url encoder

### Install ayashii-cli

```bash
curl -O https://raw.githubusercontent.com/sheepla/ayashii-cli/master/ayashii && chmod +x ayashii
```
